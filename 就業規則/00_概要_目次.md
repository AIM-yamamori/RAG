# 株式会社フィクトワークス 就業規則・ルールブック

## このドキュメント集について

本ドキュメント集は、株式会社フィクトワークス(以下「当社」)における就業規則および各種社内ルールをまとめたものです。社員はすべての項目を理解し、遵守する義務を負います。

| 会社名 | 株式会社フィクトワークス(Fictworks Inc.) |
| 設立 | 2014年4月1日 |

| 事業内容 | Webサイト・アプリケーション開発、ITコンサルティング、デジタルマーケティング支援 |
| 従業員数 | 約120名(2026年4月時点) |
| 代表者 | 代表取締役 架空 太郎 |

| 就業規則の施行日 | 2024年4月1日(最終改定:2026年4月1日) |
