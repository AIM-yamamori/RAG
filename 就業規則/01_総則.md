# 第1章 総則

## 第1条(目的)

この就業規則(以下「本規則」という)は、株式会社フィクトワークス(以下「会社」という)に勤務する従業員の労働条件、服務規律その他の就業に関する事項を定めることを目的とする。本規則に定めのない事項については、労働基準法その他関係法令の定めるところによる。

## 第2条(適用範囲)

1. 本規則は、会社に雇用される全従業員に適用する。
2. 従業員の区分は以下のとおりとする。
   - **正社員**:雇用期間を定めずに採用された従業員
   - **契約社員**:雇用期間を定めて採用された従業員
   - **パートタイム社員**:1週間の所定労働時間が正社員より短い従業員
   - **アルバイト**:学生または短期雇用を前提として採用された従業員
3. 契約社員、パートタイム社員、アルバイトの就業条件については、本規則のほか個別の雇用契約書および「有期雇用社員就業規程」の定めによるものとし、本規則と異なる場合は個別契約を優先する。

## 第3条(従業員の心構え)

従業員は、会社の経営理念「テクノロジーで顧客の挑戦を後押しする」を理解し、誠実かつ公正に職務を遂行しなければならない。

## 第4条(規則の遵守義務)

会社および従業員は、ともに本規則を守り、相互に協力して業務の運営に努めなければならない。

## 第5条(用語の定義)

本規則において使用する用語の定義は以下のとおりとする。

| 用語 | 定義 |
|---|---|
| 所定労働時間 | 第3章で定める、従業員が労働すべき時間 |
| 所定休日 | 会社が定める休日(土曜・日曜・祝日・年末年始等) |
| 法定休日 | 労働基準法で定める週1日以上の休日 |
| 管理監督者 | 労働基準法第41条に定める管理監督的地位にある者(部長職以上を原則とする) |
| みなし労働時間制 | 実際の労働時間にかかわらず、あらかじめ定めた時間を労働したものとみなす制度 |
| テレワーク | 自宅その他会社が認めた場所において情報通信機器を利用して行う業務 |

## 第6条(規則の変更)

会社は、必要に応じて本規則を変更することができる。変更にあたっては、労働基準法の定めに従い、従業員代表の意見を聴取し、所轄労働基準監督署に届け出るものとする。変更後の規則は、社内ポータルへの掲示その他の方法により周知する。

## 第7条(規則の保管・閲覧)

本規則は人事総務部にて保管し、従業員はいつでも人事総務部または社内ポータル(イントラネット内「規程集」フォルダ)にて内容を確認できる。
