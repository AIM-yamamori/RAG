# 第5章 給与・賞与

## 第36条(給与の構成)

従業員の給与は、以下の項目により構成される。

| 構成項目 | 内容 |
|---|---|
| 基本給 | 職務遂行能力・経験・職務内容等に応じて個別に決定 |
| 役職手当 | 役職に応じて支給(下表参照) |
| 住宅手当 | 月額最大20,000円(賃貸契約者のみ、世帯主に限る) |
| 通勤手当 | 実費(上限月額35,000円) |
| テレワーク手当 | 月額3,000円(テレワーク利用者) |
| 資格手当 | 会社指定の資格取得者に支給(下表参照) |
| 残業手当 | 法定の割増賃金を支給 |
| 家族手当 | 配偶者:月額10,000円、子1人につき月額5,000円(上限3人まで) |

## 第37条(役職手当)

| 役職 | 月額手当 |
|---|---|
| 部長 | 80,000円 |
| 課長 | 50,000円 |
| 係長・チームリーダー | 25,000円 |

## 第38条(資格手当)

会社が指定する以下の資格を取得した従業員に対し、資格手当を支給する(取得後申請のあった月の翌月から支給開始)。

| 資格 | 月額手当 |
|---|---|
| AWS認定資格(アソシエイト以上) | 5,000円 |
| 基本情報技術者試験 | 3,000円 |
| 応用情報技術者試験 | 5,000円 |
| 情報処理安全確保支援士 | 10,000円 |
| TOEIC730点以上 | 5,000円 |
| PMP(Project Management Professional) | 10,000円 |

## 第39条(給与の計算期間および支払日)

1. 給与の計算期間は、毎月1日から末日までとする。
2. 給与は、当月末日締め、**翌月25日**に従業員が指定する金融機関口座へ支払う。25日が休日にあたる場合は、その前営業日に支払う。

## 第40条(給与からの控除)

会社は、給与支払いの際、以下のものを控除する。

- 所得税、住民税
- 健康保険料、介護保険料、厚生年金保険料、雇用保険料(従業員負担分)
- 社内預金、社員持株会会費等、従業員が同意したもの

## 第41条(時間外労働等に対する割増賃金)

| 種別 | 割増率 |
|---|---|
| 法定時間外労働(残業) | 25% |
| 法定時間外労働が月60時間超の部分 | 50% |
| 法定休日労働 | 35% |
| 深夜労働(22:00〜5:00) | 25%(他の割増と合算) |

## 第42条(欠勤・遅刻・早退時の給与)

欠勤、遅刻、早退があった場合は、その時間に応じて給与を控除する(ノーワーク・ノーペイの原則)。計算方法は「月給÷該当月の所定労働時間×不就労時間」とする。

## 第43条(賞与)

1. 会社は、原則として年2回(7月・12月)、会社の業績および個人の評価に応じて賞与を支給する。
2. 賞与の支給は、支給日に在籍している従業員に対して行う(算定対象期間に在籍していても、支給日に退職している場合は支給しない)。
3. 賞与額は固定ではなく、会社業績および個人の人事評価により変動する。

## 第44条(昇給)

昇給は、原則として年1回(4月)、人事評価に基づき行う。会社の業績によっては昇給を見送ることがある。

## 第45条(給与の改定・人事評価)

1. 人事評価は年2回(上期:4月〜9月、下期:10月〜3月)実施する。
2. 評価結果は、昇給・賞与・昇格の判断材料として用いる。
3. 評価制度の詳細は別途「人事評価ガイドライン」に定める。
