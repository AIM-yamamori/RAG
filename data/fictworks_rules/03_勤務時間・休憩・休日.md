# 第3章 勤務時間・休憩・休日

## 第17条(所定労働時間)

1. 従業員の所定労働時間は、1週間40時間、1日8時間とする。
2. 標準の勤務時間は以下のとおりとする(部署により異なる場合は所属長が別途通知する)。

| 項目 | 時間 |
|---|---|
| 始業時刻 | 9:00 |
| 終業時刻 | 18:00 |
| 休憩時間 | 12:00〜13:00(1時間) |
| 実労働時間 | 8時間 |

## 第18条(フレックスタイム制)

1. 会社は、エンジニア職・デザイナー職を中心にフレックスタイム制を導入している。対象者は所属長の承認を得て適用される。
2. フレックスタイム制における各種時間帯は以下のとおりとする。

| 区分 | 時間帯 |
|---|---|
| コアタイム(必須勤務時間帯) | 11:00〜15:00 |
| フレキシブルタイム(早朝) | 6:00〜11:00 |
| フレキシブルタイム(夕方以降) | 15:00〜22:00 |
| 標準労働時間 | 1日8時間 |
| 清算期間 | 1か月(毎月1日〜末日) |

3. 清算期間における不足時間は翌月の労働時間に加算するか、不足分を欠勤として処理する。ただし、繰越は1か月分を上限とする。

## 第19条(休憩時間)

1. 労働時間が6時間を超える場合は45分以上、8時間を超える場合は1時間以上の休憩を、労働時間の途中に与える。
2. 休憩時間は自由に利用させるものとし、会社は休憩中の外出を制限しない。ただし、外出時はセキュリティカードを必ず携帯すること。

## 第20条(休日)

1. 会社の所定休日は以下のとおりとする。
   - 土曜日、日曜日
   - 国民の祝日
   - 年末年始(12月29日〜1月3日)
   - 夏季休暇(会社が指定する3日間。通常8月中)
   - 創立記念日(4月1日)
2. 上記のうち、**日曜日を法定休日**とする。
3. 業務上の必要がある場合、会社は休日を他の日に振り替えることができる(振替休日)。振替を行う場合は、原則として事前に従業員へ通知する。

## 第21条(休日労働・時間外労働)

1. 会社は、業務上必要がある場合、従業員に対し、所定労働時間を超える労働(残業)または休日における労働を命じることがある。
2. 残業・休日労働を命じる場合は、あらかじめ「36協定(時間外労働・休日労働に関する協定)」の範囲内で行うものとする。
3. 妊娠中の女性従業員、産後1年を経過しない女性従業員が請求した場合、残業・休日労働・深夜労働をさせない。
4. 満18歳未満の従業員には、原則として残業・休日労働・深夜労働をさせない。

## 第22条(時間外労働の手続き)

残業を行う場合、従業員は事前に勤怠管理システム(「フィクトワークス勤怠」)にて残業申請を行い、所属長の承認を得なければならない。事前申請が困難な緊急対応の場合は、事後速やかに申請・承認を得るものとする。

## 第23条(深夜業)

午後10時から午前5時までの間に労働させた場合は、深夜労働として深夜割増賃金(第5章参照)を支払う。

## 第24条(テレワーク)

1. 会社は、業務の性質上可能と判断される従業員に対し、テレワーク(在宅勤務)を認めることができる。
2. テレワークを利用する場合、原則として週3日を上限とし、事前に所属長の承認を得るものとする。
3. テレワーク中の勤怠管理は、勤怠管理システムへの打刻、および定時のオンラインミーティング(朝会・夕会)への参加により行う。
4. テレワークに必要な通信費の一部として、月額3,000円のテレワーク手当を支給する(第5章参照)。

## 第25条(出退勤の記録)

従業員は、出社・退社時に必ず勤怠管理システムへの打刻を行わなければならない。打刻を忘れた場合は、当日中に所属長へ報告し、事後修正申請を行うこと。

## 第26条(遅刻・早退・欠勤)

1. 従業員が遅刻・早退・欠勤をする場合は、事前に所属長へ連絡しなければならない。事前連絡が困難な場合は、始業時刻後できるだけ早く連絡すること。
2. 私傷病による欠勤が3日以上続く場合、医師の診断書の提出を求めることがある。
3. 無断欠勤は懲戒の対象となる(第8章参照)。
