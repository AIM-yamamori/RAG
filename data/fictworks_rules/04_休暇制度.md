# 第4章 休暇制度

## 第28条(年次有給休暇)

1. 会社は、雇入れの日から6か月間継続勤務し、全労働日の8割以上出勤した従業員に対し、年次有給休暇(以下「年休」という)を付与する。
2. 付与日数は以下のとおりとする(正社員・フルタイム勤務の場合)。

| 勤続年数 | 付与日数 |
|---|---|
| 6か月 | 10日 |
| 1年6か月 | 11日 |
| 2年6か月 | 12日 |
| 3年6か月 | 14日 |
| 4年6か月 | 16日 |
| 5年6か月 | 18日 |
| 6年6か月以上 | 20日 |

3. パートタイム社員等、所定労働日数が少ない従業員については、労働基準法に定める比例付与の方法により付与する。
4. 年休の未使用分は、付与日から2年間に限り次年度に繰り越すことができる(2年を超えた分は失効する)。
5. 年休は原則として1日単位で取得するものとするが、半日単位での取得も認める。1時間単位の取得は対象外とする。
6. 会社は、年休が10日以上付与される従業員に対し、付与日から1年以内に**最低5日**の年休を取得させなければならない(年5日の取得義務)。取得が進んでいない従業員に対しては、会社が時季を指定して取得させることがある。

## 第29条(年休の申請方法)

1. 年休を取得する場合、原則として取得希望日の3営業日前までに勤怠管理システムにて申請し、所属長の承認を得るものとする。
2. 事業の正常な運営に支障がある場合、会社は取得日の変更を求めることがある(時季変更権)。

## 第30条(特別休暇)

会社は、以下の場合に特別休暇(有給)を付与する。

| 休暇の種類 | 付与日数 | 備考 |
|---|---|---|
| 結婚休暇(本人) | 5日 | 入籍日または挙式日の前後1か月以内に取得 |
| 結婚休暇(子の結婚) | 2日 | |
| 配偶者出産休暇 | 2日 | 配偶者の出産前後8週間以内に取得 |
| 忌引休暇(配偶者・子・父母) | 5日 | |
| 忌引休暇(祖父母・兄弟姉妹・配偶者の父母) | 3日 | |
| 忌引休暇(その他の親族) | 1日 | |
| 災害休暇 | 必要な期間(上限7日) | 地震・火災等の災害で被災した場合 |
| リフレッシュ休暇 | 5日 | 勤続10年・20年・30年の節目に付与 |
| ボランティア休暇 | 年3日まで | 社会貢献活動に参加する場合 |

## 第31条(産前産後休業)

1. 会社は、出産予定の女性従業員が請求した場合、産前6週間(多胎妊娠の場合は14週間)の休業を与える。
2. 会社は、産後8週間を経過しない女性従業員を就業させない。ただし、産後6週間を経過した者が請求し、医師が支障ないと認めた業務についてはこの限りでない。

## 第32条(育児休業)

1. 従業員は、1歳に満たない子を養育するため、育児休業を取得することができる(法令に基づき、最長2歳まで延長可能な場合がある)。
2. 育児休業の申請は、原則として休業開始予定日の1か月前までに人事総務部へ申し出るものとする。
3. 育児休業中の給与は無給とするが、雇用保険からの育児休業給付金の申請をサポートする。
4. 1歳に満たない子を育てる従業員は、1日2回各30分の育児時間を請求することができる。
5. 小学校就学前の子を育てる従業員は、子の看護等を目的として、年5日(対象の子が2人以上の場合は年10日)を上限に「子の看護休暇」を時間単位で取得できる。

## 第33条(介護休業)

1. 従業員は、要介護状態にある対象家族を介護するため、対象家族1人につき通算93日まで、3回を上限として介護休業を取得することができる。
2. 要介護状態にある対象家族の介護その他の世話を行う従業員は、対象家族1人につき年5日(対象家族が2人以上の場合は年10日)を上限に「介護休暇」を時間単位で取得できる。

## 第34条(生理休暇)

生理日の就業が著しく困難な女性従業員が請求した場合、必要な期間の休暇を与える。取得日数の上限は設けない。なお、当該休暇は無給とする。

## 第35条(休暇の無給・有給の区分)

| 休暇種別 | 給与の扱い |
|---|---|
| 年次有給休暇 | 有給 |
| 特別休暇(第30条) | 有給 |
| 産前産後休業 | 無給(出産手当金で補填) |
| 育児・介護休業 | 無給(給付金で補填) |
| 生理休暇 | 無給 |
| 子の看護休暇・介護休暇 | 無給 |
